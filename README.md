# ArduinoBit

Ruby Gem for communicating with [littleBits Arduino module](http://littlebits.cc/bits/arduino).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'arduino_bit'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install arduino_bit
